# BLACK TIGER AUDIO — Investor One‑pager (GitHub Pages)

Готовый мини‑сайт/презентация (HTML слайды) для отправки инвестору.

## Как задеплоить на GitHub Pages (3 минуты)
1) GitHub → New repository (например `black-tiger-invest`)  
2) Залей файлы из этого архива в корень репозитория (нужны `index.html` и папка `assets/`)  
3) Repo → Settings → Pages  
4) Source: Deploy from a branch  
5) Branch: main, Folder: /(root) → Save  
6) Ссылка появится через минуту: `https://<username>.github.io/black-tiger-invest/`
